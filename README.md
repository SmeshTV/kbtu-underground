# KBTU Underground — Chat Application

> A real-time chat application developed during studies at Kazakh-British Technical University.

## 📋 About

This project was built as part of the academic curriculum at KBTU. It demonstrates real-time communication features and web development skills.

## 🛠️ Technologies

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Real-time:** WebSocket / Socket.io

## 🎯 Purpose

Educational project focusing on:
- Real-time data transmission
- Client-server architecture
- WebSocket protocol
- Event-driven programming
